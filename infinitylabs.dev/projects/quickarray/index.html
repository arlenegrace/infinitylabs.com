Project

Quick Array

Description

A fast C++ resizable array library with small object optimization

Built from the ground up with performance in mind, quick array is up to six times faster than GCC and Clang's std::vector implementation. It employs small object optimization, the copy and swap idiom, and other techniques to avoid heap allocations as well as saving memory when storing less than 23 bytes of data on the x86_64 platform.

Usage

Setup Guide

To use quick array, simply include quickarray.h in your source file. You can adjust certain parameters like growth factor with the macros defined at the start of the header file. Be sure to set the debug macro to false in production builds for increased performance.
